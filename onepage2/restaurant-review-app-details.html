Restaurant Review Website Details


Project Overview

I have converted a static webpage to an offline-ready web application in this project. The task was broken into three main stages:

  • Convert the page to responsive design.
  • Make the app accessible for screen reader users.
  • Add a service worker to create a seamless offline user experience.
A starter code was provided.

Implementation Details

Implementation


Convert into the Responsive Design

The website has three different layouts for three different viewport sizes: 600 px, 960 px, and 1280 px. It makes it responsive on different screen sizes. Three sets of images were used to make the website's images responsive: small, medium, and large.


Make the App accessible to Screen Readers

To make the website accessible for screen readers:

  • `Alt` attribute was added to all images.
  • Style of the links and buttons was changed so that users can spot currently focused links - `:hover` and `:focus` states have different colors.
  • A 'skip link' is provided. It allows users to skip the map and go straight to the main content.
  • Aria-labels, roles, and links added according to ARIA recommendations.


A Service Worker and Caches Added

The goal of the last stage of the project - make the website available for offline use.

  • A service worker has been added to retrieve or cache resource requests.
  • A separate cache for the website's images was created, so once an image has been cached, the service worker can respond with it the next time it's requested.
  • A warning message that the website is offline and maps cannot be displayed at the moment added.


How to Start the Website

    You need to have Python 2 or 3 installed on your computer.
  • Open a terminal window.
  • Navigate to the ./Projects/Restaurant-Review-App directory in the website's source code
  • Start a simple HTTP server in this folder to serve up the site files on your local computer. Python has some simple tools to do this.
    • If you have Python 2.x, spin up the server with `python -m SimpleHTTPServer 8000`.
    • For Python 3.x, you can use `python3 -m http.server 8000`.
  • With your server running, visit the site: `http://localhost:8000`.
